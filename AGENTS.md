# Buckeye Marketplace - AI Agent Instructions

## Project Overview

Buckeye Marketplace is a full-stack e-commerce application built with:

- **Frontend:** React + TypeScript (Vite)
- **Backend:** .NET Web API
- **Database:** SQL Server (InMemory for development)

## Architecture

- Frontend runs on http://localhost:5173
- Backend runs on http://localhost:5000
- API base path: /api/

## Frontend Conventions

- Components live in `src/components/`
- Pages live in `src/pages/`
- State management: useReducer + Context API for feature-level state
- Types/interfaces go in `src/`
- Use functional components only
- Use TypeScript types for props and state
- Use named exports for components, default export for pages

## Current State

- Product catalog: working (displays from API)
- Product detail: working
- Shopping cart: in progress

## When Generating Code

- Always include TypeScript types
- Use immutable updates
- Use aria-labels on interactive elements when appropriate
- Use useReducer for complex state and useState for simple state
